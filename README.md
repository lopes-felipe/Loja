# Loja
Projeto desenvolvido em 48 horas para um processo seletivo, através da seguinte especificação:

"Elabore uma aplicação exemplo, utilizando C#, MVC, JQUERY, AJAX, nHibernate ou EntityFramework, SQL Server, onde toda a camada de negócios fique isolada da camada de serviços (WCF) e exibição, aplicando os conceitos de orientação a objeto e DDD.
A aplicação deverá ser composta de telas de cadastro de ao menos dois tipos de objetos que se correlacionem, bem como suas telas de consulta. Fica a critério do candidato a definição do contexto (objetos) que será utilizado para a aplicação.

Todos os nomes de métodos, variáveis e classes devem estar escritos em português."

Linguagens e tecnologias utilizadas:
- C#;
- .NET Framework 4.5;
- Asp.Net MVC 5;
- WCF;
- SqlServer 2012;
- Entity Framework 6 - Database First;
- jQuery;
- Bootstrap;
- KnockoutJS;
- Ninject;
